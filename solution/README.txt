Qua ci va tutto il codice.

Suddivisione della directory:

-- Frontend
Tutto ciò che riguarda l'interfaccia web.

-- Central Server
Server centrale in express: deve essere leggero e reindirizzare a tutti gli altri server.

-- Mongo Database Server
Server che gestisce il database di mongodb.

-- Postgre Database Server
Server che gestisce il database di postgre-sql

-- Jupyter
Tutto ciò che riguarda l'analisi del dataset fornito, attraverso jupyter notebook.

-- Chat Server
Server dedicato alle chat con websocket. /// NON ESPRESSAMENTE RICHIESTO - VALUTARNE UTILIZZO ///
